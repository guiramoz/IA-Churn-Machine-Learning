# Previsão de Churn de Clientes com Inteligência Artificial

## 📖 Sobre o Projeto

Este projeto foi desenvolvido durante o curso de **Programação em IA Generativa** do SENAI. O sistema utiliza técnicas de **Inteligência Artificial** para analisar dados de clientes e prever a probabilidade de **churn** (cancelamento de serviços).

A proposta do projeto é demonstrar como modelos de IA podem identificar padrões de comportamento dos clientes, permitindo que empresas antecipem possíveis cancelamentos e adotem estratégias para aumentar a retenção e melhorar a tomada de decisões.

## 🚀 Funcionalidades

* Análise de dados de clientes.
* Previsão da probabilidade de churn.
* Identificação de clientes com maior risco de cancelamento.
* Apoio à tomada de decisões estratégicas.
* Auxílio na definição de ações para retenção de clientes.
* Simulação de um sistema de análise preditiva.

## 🛠️ Tecnologias Utilizadas

* Python
* Inteligência Artificial
* Machine Learning
* Análise de Dados

## ⚙️ Funcionamento

1. Os dados dos clientes são fornecidos ao sistema.
2. A inteligência artificial analisa as informações disponíveis.
3. O modelo identifica padrões associados ao comportamento dos clientes.
4. É calculada a probabilidade de cada cliente cancelar o serviço.
5. Os resultados podem ser utilizados para orientar estratégias de retenção e reduzir o índice de churn.

## 🎯 Objetivo

Este projeto possui finalidade acadêmica e tem como objetivo aplicar conceitos de Inteligência Artificial e Machine Learning na análise preditiva de dados, demonstrando como essas tecnologias podem auxiliar empresas na identificação de clientes com maior probabilidade de cancelamento e contribuir para decisões mais estratégicas.
